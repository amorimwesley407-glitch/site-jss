🌐 Site Institucional – Empresa de Telecomunicações (Fibra Óptica)

Este projeto é um site institucional desenvolvido para uma empresa de telecomunicações especializada na venda de serviços de internet via fibra óptica. A plataforma foi criada com foco em performance, responsividade e experiência do usuário, apresentando de forma clara os planos, benefícios e diferenciais da empresa.

🚀 Tecnologias Utilizadas

React – Utilizado para construção de componentes dinâmicos e gerenciamento de interface.

TypeScript

HTML5

CSS3

JavaScript

🎯 Funcionalidades

Página inicial moderna e responsiva

Apresentação de planos de internet fibra óptica

Seção institucional (Sobre a empresa)

Formulário de contato

Design adaptado para dispositivos móveis

Componentização para melhor manutenção e escalabilidade

💡 Objetivo do Projeto

Desenvolver uma aplicação web rápida, moderna e intuitiva, capaz de transmitir credibilidade e facilitar a captação de novos clientes para a empresa de telecomunicações.
